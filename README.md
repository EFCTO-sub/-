## 변수와 자료형

### 변수
변수는 데이터를 저장하는 메모리 공간에 붙여지는 이름으로 파이썬에서는 변수를 선언할 때 특별한 키워드 없이 = 연산자를 사용하여 값을 할당합니다.

### 자료형
자료형은 변수가 어떤 종류의 데이터를 저장하는지 나타내며 파이썬은 여러가지 내장 자료형을 제공한다

*   **정수 (Integer, `int`)**: 정숫값을 표현할수 있다
    *   예시: `10`, `-5`, `0`

*   **실수 (Float, `float`)**: 소수(실수)값까지 표현할수 있다
    *   예시: `3.14`, `-0.5`, `2.0`

*   **문자열 (String, `str`)**: 따옴표(`''` 또는 `""`)로 묶인 문자이다
    *   예시: `'안녕하세요'`, `"Python"`, `'123'`, `"""hi"""` <- 참고로 따옴표를 3개연속으로 놓으면 줄바꿈이 가능한 문자열이 된다
    *   기타: 그냥 문자열을 코드 중간에 놓으면 인터프리터는 그냥 씹어버리기때문에 주석으로 활용도 가능하다.)

*   **리스트 (List, `[]`)**: 대괄호로 여러 값을 묶어서 저장합니다
    *    예시: `[1, 2, 3, 4, 5], ["홍길동", "남", 20]`

*   **튜플 (Tuple, `()`)**: 불변형 자료형으로 리스트처럼 여러 값을 묶어 저장하나 값을 변경할수 없다
    *    예시: `(1, 2, 3)`

*   **딕셔너리 (Dictionary, `{key : value}`)**: 키:값 쌍으로 저장하는 자료형으로 순서가 없다
    *    예시: `{"사과" : "apple"}
